# TextPro-String-Analyzer
A robust java utility for real-time text analysis and transformation.
# FEATURES
- Analysis : Tracks word counts , character counts(with/without spaces) , and vowel frequency.
- Formatting : Provides Header Style , Snake_Case , and initials generation.
- Logic : Dual-layer reversal algorithm( word order and character sequence).
- Output : Generates a clear , professional terminal summary report.
# TECHNOLOGY
- Core : Java Scanner for input.
- Mthods : Implements custom loop-based iterations , substring manipulation , and array splitting for complex processing.
